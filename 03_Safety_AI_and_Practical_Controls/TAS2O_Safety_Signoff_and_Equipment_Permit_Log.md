# TAS2O Safety Sign-Off and Equipment Permit Log

This log records student acknowledgement and practical equipment permission.

Log fields:

- Student name or anonymized code
- Date of safety briefing
- 3D printer acknowledgement
- Circuit board acknowledgement
- AI tool access acknowledgement, if applicable
- Teacher signature
- Permission granted or withheld
- Follow-up notes, if required

Procedure:

- Review the relevant safety rules before the first practical lesson.
- Obtain student acknowledgement before any independent equipment use.
- Record any restricted access, extra support, or retraining requirement.
- File incident notes separately in the safety package if a concern arises.

Filing guidance:

- Keep this log with the safety package and practical resources addendum.
- Add the log to the inspection binder only after it contains live course entries.
