# TAS2O Facilities Resources Addendum

This addendum is filed with the facilities inventory, maintenance, and setup records.

Confirmed school resources for TAS2O delivery:

- 3D printer for rapid prototyping and iterative design work
- Simplified circuit boards for introductory electronics and testing
- AI API access for supervised classroom experimentation

Facilities filing guidance:

- Record the 3D printer in the equipment inventory with model, location, maintenance date, material type, and operating status.
- Record the simplified circuit boards in the materials and consumables log with quantities, replacement cycle, and storage location.
- Record the AI API access in the technology setup checklist with the account owner, access method, supervision rule, and logging requirement.
- If the printer or circuit boards require calibration or inspection, record the result in the maintenance log before the course begins.
- If AI access is enabled, note the approved use case and the restriction against unsupervised student credential sharing.
