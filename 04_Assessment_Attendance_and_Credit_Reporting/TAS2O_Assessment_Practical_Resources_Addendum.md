# TAS2O Assessment and Practical Resources Addendum

This addendum links the live practical resources to attendance, assessment, and lesson evidence.

Assessment evidence expectations:

- 3D printer:
  - Include design iteration sketches, print settings, failed prototypes, revisions, and final product photos
  - Log the lesson in the daily lesson record and identify the practical task completed
  - If a student misses the practical, record the make-up plan and any alternate evidence
- Simplified circuit boards:
  - Include build diagrams, test results, troubleshooting notes, and teacher feedback on safe operation
  - Connect observations to the assessment records package and observation record form
  - Keep a clear record of which students completed which circuit task
- AI API access:
  - Record when AI assistance was allowed, what the prompt/output workflow was, and how the student verified the result
  - Keep prompt/output evidence only when it is permitted by the AI policy and privacy rules
  - Mark any assessment where AI support changes the authenticity or weighting of the evidence

Recommended filing:

- Daily lesson logs should name the practical resource used.
- Attendance logs should show whether the practical was completed or made up.
- Student conference records should reference the practical task and the teacher feedback given.
