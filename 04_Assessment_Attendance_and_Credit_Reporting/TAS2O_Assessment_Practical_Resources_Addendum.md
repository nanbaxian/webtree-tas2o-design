# TAS2O Assessment and Practical Resources Addendum

This addendum links the practical resources to attendance, assessment, and lesson evidence.

Assessment evidence:

- 3D printer:
  - Include design sketches, print settings, failed prototypes, revisions, and final product photos.
  - Record the lesson in the daily lesson log and identify the practical task completed.
  - If a student misses the practical, record the make-up plan and any alternate evidence.
- Simplified circuit boards:
  - Include build diagrams, test results, troubleshooting notes, and teacher feedback on safe operation.
  - Connect observations to the assessment records package and observation record form.
  - Keep a clear record of which students completed each circuit task.
- AI API access:
  - Record when AI assistance was permitted, the prompt/output workflow, and how the student verified the result.
  - Keep prompt/output evidence only when permitted by the AI policy and privacy rules.
  - Flag any assessment where AI support affects the authenticity or weighting of the evidence.

Filing guidance:

- Daily lesson logs should identify the practical resource used.
- Attendance logs should show whether the practical was completed or made up.
- Student conference records should reference the practical task and the teacher feedback provided.
- If moderation evidence is required, attach the anonymous assessment sample packet to the same assignment record.
