# TAS2O Daily Lesson Log Completion Guide

This guide explains how the daily lesson log should be completed after the course begins.

Required entries for each lesson:

- Date
- Period or class section
- Lesson focus
- Practical resource used, if any
- Attendance summary
- Homework or next-step task
- Notes on student support, safety, or make-up work

Link to the 110-hour plan:

- Record each lesson against the pacing guide.
- Show that the total logged instructional time matches the course delivery plan.
- Note any cancellations, substitutions, or make-up hours.

Filing guidance:

- Keep the log current during the course, not only at the end.
- Use the same course section naming across all attendance and assessment records.
- If a lesson includes practical work, identify the equipment or tool used.
