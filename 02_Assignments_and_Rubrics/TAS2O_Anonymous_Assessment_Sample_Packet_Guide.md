# TAS2O Anonymous Assessment Sample Packet Guide

This guide defines the anonymized sample packet to be added once real student work is available.

Required contents for one task:

- One anonymized student submission for the same assignment used by the class.
- Teacher annotations showing strengths, gaps, and the reasons for the assigned level.
- A completed rubric with category-by-category marks or descriptors.
- A short moderation note describing how the sample supports consistent application of the rubric.

Filing guidance:

- Use one complete sample from a mid-range student first, then add a second sample if moderation is required.
- Remove names, student numbers, and identifying details before filing.
- Keep the sample paired with the assignment, rubric, and teacher exemplar for the same task.

Operational note:

- Do not create fictional student work for this packet.
- Add the packet only after real work has been produced in class.
