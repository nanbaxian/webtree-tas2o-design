# TAS2O Inspection Resources Navigation Addendum

This addendum tells an inspector where to find the evidence for the three practical resources.

Inspection navigation:

- 3D printer:
  - Facilities inventory and maintenance log
  - Lab/classroom setup checklist
  - Safety package
  - Assessment records showing practical design work
- Simplified circuit boards:
  - Materials and consumables budget
  - Lab/classroom setup checklist
  - Safety package
  - Observation records and student conference notes
- AI API access:
  - Technology setup and account access checklist
  - AI policy and academic integrity package
  - Daily lesson logs where AI use is supervised and disclosed
  - Assessment records showing how AI-assisted work was checked

Binder guidance:

- Keep this addendum behind the inspection binder index so the inspector can move directly from the checklist to the supporting evidence.
- If the school changes equipment, access rules, or vendor accounts, update the navigation addendum first and then the linked records.
